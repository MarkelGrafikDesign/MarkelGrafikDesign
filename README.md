# Lisa Markel – Graphic Design Portfolio

Persönliches Portfolio von Lisa Markel, Grafikdesignerin mit Schwerpunkt auf Print- und Digital-Layout, Brand Identity und Illustration.

**Live-Ansicht:** [lisamarkel.github.io/portfolio](#) *(Link anpassen, sobald GitHub Pages aktiv ist)*

## Über dieses Projekt

Das Portfolio ist als **einzelne, in sich geschlossene HTML-Datei** aufgebaut – keine externen Assets, keine Build-Tools, keine Abhängigkeiten. Alle Bilder sind als Base64 direkt eingebettet. Das macht die Datei groß, aber komplett portabel: ein Download, ein Upload, fertig.

- **Struktur:** Über mich · Projekte · Eigene Kunst · Kontakt
- **Design:** Dunkles Navy/Gold auf Parchment-Ton, `Playfair Display` für Headlines, `Inter` für Fließtext, `DM Mono` für Labels/Meta-Infos
- **Technik:** reines HTML + CSS, keine JavaScript-Frameworks

## Lokal ansehen

Repository klonen und `index.html` direkt im Browser öffnen – kein Server nötig:

```bash
git clone https://github.com/<username>/<repo>.git
cd <repo>
open index.html   # macOS
# oder: start index.html   (Windows)
# oder: xdg-open index.html   (Linux)
```

## Mit GitHub Pages veröffentlichen

1. Repository-Einstellungen → **Pages**
2. Branch `main`, Ordner `/ (root)` auswählen
3. Speichern – die Seite ist nach kurzer Zeit unter `https://<username>.github.io/<repo>/` erreichbar
